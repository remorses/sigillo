# Sigillo — Visual Identity

## Style Prompt

Dark cinematic security aesthetic inspired by Shadow Cut. Deep blacks with a single toxic-green accent evoking terminal cursors and encrypted data. Sharp angular type, dramatic reveals from darkness. Content emerges from the void — the reveal IS the story. Sigillo protects secrets, and the visual language should feel like a vault door opening.

## Colors

- **Background:** `#0a0a0f` — near-black, the void
- **Foreground:** `#e8e8ec` — cool white for primary text
- **Accent:** `#39FF14` — toxic green, terminal / encrypted-data energy
- **Muted:** `#3a3a44` — cold grey for secondary elements
- **Danger:** `#C1121F` — blood red for leaked-secret warnings

## Typography

- **Headlines:** Inter, 800 weight, tight letter-spacing (-0.03em)
- **Body / UI:** Inter, 400–500 weight

## What NOT to Do

- No gradient text or gradient backgrounds — solid colors only
- No rounded pastel cards — this is a security tool, not a wellness app
- No cyan-on-dark neon palette — we use toxic green, not electric blue
- No centered-everything layouts — lead the eye with asymmetry
- No decorative flourishes — every element earns its place
